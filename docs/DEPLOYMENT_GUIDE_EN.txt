ISSUE: "Cannot find app.py"
FIX:
  1. Verify app.py is in mc_sito_web root
  2. Check render.yaml paths are correct
  3. Settings â†’ Trigger Deploy â†’ Manual Deploy

ISSUE: Static files (CSS/JS) return 404
FIX:
  1. Check browser Console (F12)
  2. Verify app.py has static routes
  3. Rebuild + redeploy

ISSUE: "ModuleNotFoundError"
FIX:
  1. Verify requirements.txt has all packages
  2. Check Logs for missing imports
  3. Add missing package to requirements.txt

ISSUE: App slow after inactivity
FIX:
  1. Normal for free tier (auto-sleep)
  2. First request after sleep = 30sec delay
  3. Upgrade to PRO to remove this

ISSUE: Database locked
FIX:
  1. Free tier uses /tmp (temporary)
  2. For production: upgrade to PostgreSQL
  3. Or: just restart the service
